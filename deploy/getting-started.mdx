## Node Package

### [NPM](https://www.npmjs.com/package/alchira-central)

- By default Alchira uses npm for package distribution.

```bash
# Local Installation
npm install -g alchira-central
# Local Command Run
alchira {command}

# Global Installation
npm install --save-dev alchira-central
# Global Command Run
npm run alchira {command}
```
