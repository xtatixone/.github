## Node Package

### [NPM](https://www.npmjs.com/package/xtatix-central)

- By default Xtatix uses npm for package distribution.

```bash
# Local Installation
npm install -g xtatix-central
# Local Command Run
xtatix {command}

# Global Installation
npm install --save-dev xtatix-central
# Global Command Run
npm run xtatix {command}
```
